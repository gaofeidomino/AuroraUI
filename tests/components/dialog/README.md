# Dialog Component Tests

This directory contains tests for the Dialog components.

## Test Files

- `Dialog.test.ts` - Tests for the main Dialog component

## Test Coverage

The tests cover:

### Dialog Component

- Basic rendering and props
- Data slot attributes
- Props forwarding
- Empty content handling

## Running Tests

To run these specific tests:

```bash
pnpm test tests/components/dialog/
```

To run with coverage:

```bash
pnpm test:coverage tests/components/dialog/
```
