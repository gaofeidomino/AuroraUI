# Command Component Tests

This directory contains tests for the Command components.

## Test Files

- `Command.test.ts` - Tests for the main Command component

## Test Coverage

The tests cover:

### Command Component

- Basic rendering and props
- Custom class application
- CSS class application
- Data slot attributes
- Props forwarding
- Empty content handling

## Running Tests

To run these specific tests:

```bash
pnpm test tests/components/command/
```

To run with coverage:

```bash
pnpm test:coverage tests/components/command/
```
