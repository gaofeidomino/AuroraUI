# DropdownMenu Component Tests

This directory contains tests for the DropdownMenu components.

## Test Files

- `DropdownMenu.test.ts` - Tests for the main DropdownMenu component

## Test Coverage

The tests cover:

### DropdownMenu Component

- Basic rendering and props
- Data slot attributes
- Props forwarding
- Empty content handling

## Running Tests

To run these specific tests:

```bash
pnpm test tests/components/dropdown-menu/
```

To run with coverage:

```bash
pnpm test:coverage tests/components/dropdown-menu/
```
