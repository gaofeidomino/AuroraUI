# ContextMenu Component Tests

This directory contains tests for the ContextMenu components.

## Test Files

- `ContextMenu.test.ts` - Tests for the main ContextMenu component

## Test Coverage

The tests cover:

### ContextMenu Component

- Basic rendering and props
- Data slot attributes
- Props forwarding
- Empty content handling

## Running Tests

To run these specific tests:

```bash
pnpm test tests/components/context-menu/
```

To run with coverage:

```bash
pnpm test:coverage tests/components/context-menu/
```
