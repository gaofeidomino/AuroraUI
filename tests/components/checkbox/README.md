# Checkbox Component Tests

This directory contains tests for the Checkbox component.

## Test Files

- `Checkbox.test.ts` - Tests for the main Checkbox component

## Test Coverage

The tests cover:

- Basic rendering and props
- Custom class application
- Checkbox indicator rendering
- Default and custom slot content
- Data slot attributes
- State handling (checked, unchecked, indeterminate)
- Accessibility attributes
- CSS class application
- Disabled state
- Event handling (if applicable)

## Running Tests

To run these specific tests:

```bash
pnpm test tests/components/checkbox/
```

To run with coverage:

```bash
pnpm test:coverage tests/components/checkbox/
```
