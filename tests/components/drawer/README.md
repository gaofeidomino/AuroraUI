# Drawer Component Tests

This directory contains tests for the Drawer components.

## Test Files

- `Drawer.test.ts` - Tests for the main Drawer component

## Test Coverage

The tests cover:

### Drawer Component

- Basic rendering and props
- Data slot attributes
- Props forwarding
- Empty content handling
- Default prop values

## Running Tests

To run these specific tests:

```bash
pnpm test tests/components/drawer/
```

To run with coverage:

```bash
pnpm test:coverage tests/components/drawer/
```
